Frequently Asked

Does it need Screen Recording permission? +
Yes — but only to read macOS's own recording indicator, the system signal that tells Curtain whether a capture is in progress. It never sees, records, or stores your screen content. It's the same permission Apple requires for any app that needs to know capture state, and it's the only way to detect sharing reliably across every app.
Does it work with Zoom, Meet and Teams? +
Yes — all of them. Because Curtain reads the system's own capture indicator rather than hooking into individual apps, it works with Zoom, Google Meet, Microsoft Teams, Discord, Slack huddles, QuickTime, and even iPhone Mirroring. If macOS knows you're sharing, so does Curtain.
Does it detect screen recording too? +
Yes. Recording uses the exact same system capture indicator as screen sharing, so Curtain protects you whenever you record your screen — with QuickTime, a screen recorder, or anything else — just as it does during a live share.
